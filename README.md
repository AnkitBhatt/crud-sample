## app
1. I have used postgres database for crud operation.
2. Please initialize project with npm install
## Installation

```bash
$ npm install
```
3. Please use your own credentials in .env file. I have kept mine as of now as it is because it is not that confidential.
4. I have made all the API with the help of TYPEORM module
5. Please refer files which I have inserted in my database(e.g. Few minor changes as per my knowledge)
  - In Product Entity last "Image" column is not inserted by me
  - In Options Entity "values" column is not inserted by me
  - In Variants Entity "presentment_prices" column is not inserted by me
  - Thus I have made those changes and attached the same with this repository("./Testing-JSON/single-product.json")
